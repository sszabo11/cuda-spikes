# Bio-Inspired Spiking Neural Network in C with Cuda.
Trying to develop novel architecture and explore how intelligence works.

## Goal
- Build brain like network which can "wire" to any sense and produce any action
- Multi-modal system that mirrors how the brain works
- **Wire-to-brain**: Give any *sense* by encoding to spike train. 
- Network should adapt and mould itself to the stimuli and actions
- Continously learn online.
